About the Microsoft 365 NL UserGroup

The Microsoft 365 NL Usergroup is the Dutch user group for everything related to Microsoft 365.
The Modern Workplace is hotter than ever, with Microsoft leading the way. What makes Microsoft 365 hard to grasp, is the fact that is built upon several different components. Azure Active Directory, the Office 366 suite of apps, including things like Exchange Online, Teams and the PowerPlatform, Intune, Windows Virtual Desktop… There are a lot of different techniques that make up a real modern workplace. With this usergroup, we try to inform you about all these different components. From an IT-Pro point of view, with technical deep-dives, but also from an end-user point of view, trying to help you in being more productive in your day-to-day work.

The Microsoft 365 NL UserGroup is founded by three technology enthousiasts, all (of course) based in The Netherlands.

Jeroen Burgerhout

He is specialized in Azure, Azure AD, Microsoft Endpoint Management (aka Intune), Microsoft365, Windows 10, and Windows Server. In 2020, Jeroen published a Microsoft Exam MD-100 Windows 10 Certification Guide, which is available through Packt or Amazon.
